# AI Group Lunch Planner (Swiggy MCP)

This project is an AI-powered group food ordering agent built using Swiggy MCP APIs.

## Idea
The system takes group food preferences (cuisine, dietary choices, budget) and generates an optimized food order recommendation.

## Planned Features
- Preference understanding using LLM
- Integration with Swiggy Food MCP
- Restaurant and menu search
- Optimized group order suggestions

## Status
🚧 Initial development phase — MCP integration in progress
